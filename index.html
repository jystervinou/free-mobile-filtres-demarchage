Bookmarklet pour Bloquer les Appels de démarchage sur une ligne Free Mobile

Ce bookmarklet ajoute des règles qui bloquent les appels entrants à partir de numéros commençant par ces préfixes officiels :

  • 0162
  • 0163
  • 0270
  • 0271
  • 0377
  • 0378
  • 0424

  • 0568

  • 0948
  • 0949
  • 09475
  • 09476
  • 09477
  • 09478
  • 09479

Ceci est spécifique à Free Mobile et Free La Réunion.

Pour utiliser le bookmarklet, suivez ces étapes :

  1. Installez le bookmarklet :
    • Glissez-déposez le lien ci-dessous dans la barre de favoris de votre navigateur.
    • Donnez-lui un nom (par exemple, Bloquer les Appels).
  2. Connectez-vous sur https://mobile.free.fr (ou https://mobile.free.re)
  3. Activez les "Filtres Appels & SMS/MMS" sur la page des Services si ce n'est pas déjà fait
  4. Accédez à https://mobile.free.fr/account/mes-services/filtres.
  5. Cliquez sur le bookmarklet que vous avez installé pour ajouter tous les filtres des préfixes des numéros de démarchage officiels.
  6. Attendez quelques secondes puis rechargez la page des filtres.
